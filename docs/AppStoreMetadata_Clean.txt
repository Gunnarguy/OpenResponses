APP NAME
OpenResponses

SUBTITLE
Native GPT-5.2 Client

PROMOTIONAL TEXT
GPT-5.2 on iOS. Web Search, Image Generation, and full API control—bring your own key.

DESCRIPTION
A native iOS client for GPT-5.2 and reasoning models. Bring your own API key, stream responses in real time, and access built-in tools—all in a fast, privacy-focused app.

MODELS
• GPT-5.2, GPT-5.2-pro, GPT-5.1, GPT-5, and GPT-5-mini
• o3 and o4-mini reasoning models with configurable effort
• GPT-4.1 family, GPT-4o, and legacy fallbacks

BUILT-IN TOOLS
• Web Search: Live answers with clickable citations
• Image Generation: Create visuals with gpt-image-1
• Code Interpreter: Run Python in a sandboxed cloud environment
• File Search: Query vector stores and uploaded documents

STREAMING & INSPECTION
• Token-by-token streaming with reasoning indicators
• Request Inspector shows every API payload and tool call
• Local conversation history with export support

PRO CONTROLS
• Temperature, max tokens, reasoning effort, and tool routing
• Prompt Library to save and reuse configurations
• Model compatibility guidance built in

APPLE INTEGRATION
• Access Calendar, Contacts, and Reminders via AI tools
• SwiftUI interface with Dark Mode and Dynamic Type
• VoiceOver and reduced motion fully supported

PRIVACY
• API keys stored in the iOS Keychain—never leave your device
• No accounts, no analytics, no tracking
• All data stays local unless you export it

AI output can be inaccurate—always verify important information.

REQUIREMENTS
• iOS 17.0 or later
• Your own API key with active billing

APP REVIEW INFORMATION - REVIEW NOTES
The app does not request camera, microphone, speech recognition, or precise location permissions in this release.

Reviewer Guide:
1. Launch OpenResponses and paste an API key into Settings > Credentials (use the App Review note key if provided, or your own key).
2. Open Settings > Model and select "GPT-5.2" or "GPT-4.1".
3. Send "Summarize the WWDC 2025 keynote" to verify streaming responses.
4. Enable Web Search under Settings > Tools and ask "What happened in tech news today?"
5. Attach a PDF from the Files picker and ask "Summarize the attached document."

Note: A ChatGPT Plus subscription does not cover API usage. You must use a platform.openai.com API key with active billing.
