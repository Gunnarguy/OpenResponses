OpenResponses is a native iOS client for the OpenAI Responses API. Bring your own API key, stream answers in real time, inspect every request, and use built-in tools in a fast, transparent app designed for power users.

Model Coverage

- GPT-5.4, GPT-5.3, GPT-5.2, GPT-5.1, GPT-5, and GPT-5-mini
- o3 and o4-mini reasoning models with configurable effort
- GPT-4.1 family, GPT-4o, and legacy fallbacks

Built-in Tools

- Web Search with live sources and citations
- Image Generation with gpt-image-1
- Code Interpreter for Python, analysis, and charts
- File Search for uploaded files and vector stores
- MCP integrations for connected tools and remote servers

Streaming & Control

- Token-by-token streaming with reasoning indicators
- Request Inspector shows API payloads, tool calls, and responses
- Prompt Library for reusable configurations
- Temperature, reasoning effort, max tokens, and tool routing controls
- Local conversation history with export support

Apple Integration

- Calendar, Contacts, and Reminders tools
- Native SwiftUI experience with Dark Mode, Dynamic Type, and VoiceOver support

Privacy

- Your OpenAI API key is stored in the iOS Keychain
- Conversation history stays on device unless you choose to send or export it
- No required account

AI output can be inaccurate or outdated. Verify important information before acting.

Requirements

- iOS 17.0 or later
- Your own OpenAI API key with active billing
